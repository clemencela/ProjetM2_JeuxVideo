Notre projet : "Les jeux vidéos en chiffres"

+ d'informations

Dans le cadre du cours "Documents structurés" de deuxième année de master, nous avons pu mener un projet sur le thème de notre choix. Passionnées par les jeux vidéos, nous avons décidé de consacrer notre projet à cet univers. Vous trouverez tout d'abord sur ce site une présentation générale de notre démarche adoptée dans ce projet. Vous pourrez par ailleurs naviguer sur les différentes pages de résultats afin de visualiser les différentes statistiques obtenues à partir de nos données. Enfin, vous trouverez également une page "Remarques" contenant notamment le lien vers notre Github.

Bonne lecture !
